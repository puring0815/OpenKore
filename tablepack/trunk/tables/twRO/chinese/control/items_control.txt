# 本檔案可以讓你控制哪些物品要放進倉庫裡、哪些要賣掉、或是帶在身上。
#
# 格式：
# (物品名稱) (身上數量) (是否自動存倉) (是否賣掉) [放進手推車] [從手推車拿出]
# 物品名稱		：物品道具的名稱。
# 身上數量		：你希望帶在身上的最小數量。
# 是否自動存倉	：設定 1 將物品欄的該物品存倉。
#                 設定 2 將物品欄及手推車內的該物品存倉。 
# 是否賣掉		：設定 1 以賣給買物/賣物的 NPC。
# 放進手推車	：設定 1 以自動將該物品放進手推車。
# 從手推車拿出	：設定 1 以自動從手推車拿出該物品。
#
# 以下的範例會賣掉所有的傑勒比結晶、將所有的短劍存倉、並將所有的花放進手推車。
#	傑勒比結晶 0 0 1
#	短劍 0 1 0
#	花 0 0 0 1
#
# 以下的範例當身上帶的紅色藥水少於 25 瓶的時候，會從手推車中拿出紅色藥水，令身上的紅色藥水數量為25。
# 	紅色藥水 25 0 0 0 1
#
# 物品是大小寫不分的（英文）。只有在這個檔案所列的物品會被存倉或賣出。
#
# 假如 config 檔的自動存倉與自動賣物都被設為 1，OpenKore 則會先進行存倉然後再進行賣物（除非倉庫滿了）。
#
# OpenKore 不會把已裝備的物品賣出或存倉。舉例來說，假如你正裝備著短劍，
# 而你設定賣出所有的短劍，除了裝備著的那短劍外，OpenKore 會賣出身上的所有短劍。
#
# 別忘了在 config 檔設定 storageAuto、sellAuto 與其它相關的設定，否則這個檔案將不會有作用。
