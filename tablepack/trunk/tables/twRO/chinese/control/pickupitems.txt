# 本檔案設定讓 Kore 要撿取（或不撿）的物品。
#
# 格式：
# 物品名稱 <flag>
#
# 物品名稱： 物品的名稱，對照 /tables/twRO/items.txt。
# <flag>：	
#           -1 表示當它出現在物品欄時就自動丟棄（對流氓的自動偷竊很有用）。
#            0 表示不撿該物品。
#            1 表示撿取該物品。
#            2 表示用最快速度去撿，就算正被怪物圍毆，死也要撿（貴重物品可設）。
#
# 說明：
# 1.# 字符號開頭為註解，該行不被理會。
# 2.有洞物品於物品名稱後加 [洞數]，例如：笨拙短劍 [4]，雙刃短劍 [3]。
#
#### 以下為所有本檔案未提及的物品會自動撿取 ####
all 1
