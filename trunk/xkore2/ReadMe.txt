HOW TO USE Xkore2
1.Place local.xml in a folder named 'data' (create one if it doesn't exist) in the Ragnarok Online directory
2.Create a shortcut to your ragexe.exe with the /account:local.xml command
3."DRIVENAME:\RO_Directory\Ro_Client_FileName.exe" /account:local.xml

-------------------
Proxy Version
-------------------
4.Edit the server.txt in your tables directory put in:

[Xkore2]
ip 127.0.0.1
port 6900
master_version 18
version 0
serverType 0

5.Execute Kore.select Xkore2 as the server
6.Wait till you are fully logged in.
7.Execute the Shortcut.
8.Login
8.Enjoy
